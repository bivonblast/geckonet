Geckoboard.Net

A comprehensive C# API wrapper library for accessing Geckoboard.com, using XML or JSON to read/write.

Welcome to GeckoBoard.Net

Geckoboard.Net is an easy solution for adding Geckoboard endpoints to your MVC/WebAPI project. To install using NuGet in your project, just type:

PM> Install-Package geckonet
